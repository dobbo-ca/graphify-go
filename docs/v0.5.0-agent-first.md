# v0.5.0 — Agent-first delta

graphify-go is repositioned as an **agent-first** code-knowledge-graph CLI: a tool
AI agents run and parse output from, deliberately dropping human-facing
visualization. This release closes the agent-relevant gaps against upstream
python graphify and removes human-only viewers.

Work is tracked in beads (`bd list`). Each item below lands as its own PR against
this integration branch and is reviewed for security, stability, performance, and
simplicity before merge. This is the umbrella PR; review it before merge to main.
